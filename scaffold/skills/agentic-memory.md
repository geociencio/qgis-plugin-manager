# Skill: Agentic Memory (Cerebro)

Esta habilidad permite al agente gestionar su propia memoria semántica en este proyecto, extrayendo lecciones, patrones y preferencias para mejorar la efectividad a largo plazo.

## Directrices de Extracción

El agente debe buscar activamente los siguientes elementos durante la interacción:

1.  **Patrones de Error**: Soluciones a bugs complejos o recurrentes.
2.  **Preferencias del Usuario**: Estilo de código, arquitectura preferida y herramientas.
3.  **Decisiones Técnicas**: Justificaciones de implementaciones específicas.

## Protocolo de Actualización (`.agent/memory/AGENT_LESSONS.md`)

Al final de cada sesión, el agente debe:

1.  **Sintetizar**: Resumir hallazgos en entradas concisas.
2.  **Estructurar**: Usar el formato YAML definido en la plantilla de memoria.
